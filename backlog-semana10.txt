BACKLOG SEMANA 10

Las siguientes historias de usuario están organizadas de mayor a menor prioridad.

PRIORIDAD ALTA

1. Como ciudadano quiero registrar un problema de mi comunidad para dejar evidencia y permitir que el equipo encargado pueda revisarlo.

2. Como coordinador quiero recibir una alerta cuando se registre un nuevo reporte para poder revisarlo y darle seguimiento oportunamente.


PRIORIDAD MEDIA

3. Como miembro del equipo quiero consultar los reportes registrados para conocer su estado y hacer seguimiento a cada caso.

4. Como coordinador quiero actualizar el estado de un reporte para que el equipo pueda conocer los avances realizados.


PRIORIDAD BAJA

5. Como ciudadano quiero consultar el estado de mi reporte para saber si el problema ya fue revisado o solucionado.
